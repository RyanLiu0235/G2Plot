## 何时使用

日历图是通过一个日历形态的图形来显示数据在一个日期范围内的对比情况。

## 数据类型

| 适合的数据       |「一个日期字段」+「一个连续数据字段」 |
| ---------------- | ----------------------------------------------------------------- |
| 功能             | 观察数据在日期范围内的分布和对比                                                |
| 数据与图形的映射 | 一个连续字段映射图形的颜色                                  |
| 适合的数据条数   | 大量数据                                                          |
